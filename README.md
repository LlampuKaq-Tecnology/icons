## use this library

```bash
npm i @llampukaq/icons
```

```tsx
import { Icons } from "@llampukaq/icons";
export default function Home() {
  return (
    <>
      <Icons className="text-white" icon={"Icon123"} size={40} />
    </>
  );
}
```

This library use tabler icons (more information)[https://tabler-icons.io/]
